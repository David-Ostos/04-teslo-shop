# Teslo API Documentation

1. Clonar proyecto

2. ```npm i```

3. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```

4. Cambiar las variables de entorno

5. levantar db
```bash
docker compose up -d
o
docker-compose up -d 
```

6. Levantar App
```bash
npm run start:dev
```
7. Ejecutar SEED 
```
http://localhost:3000/api/seed
```
